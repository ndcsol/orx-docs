# Hotel Pricing

This page covers retrieving confirmed pricing for specific hotel rates.

## Hotel Rate Pricing

`GET /hotel/[sessionId]/pricing/[rateId]`

This endpoint retrieves confirmed pricing information for a specific hotel rate. The `rateId` parameter should be the public identifier from an Enhanced Hotel Rate Info object obtained from the room details endpoint.

> ðŸ›Ž Use this endpoint to get final confirmed pricing before proceeding to booking. Pricing may differ from room details due to availability changes.

### Request

`GET /hotel/[sessionId]/pricing/[rateId]`

### Response

Returns a single [Enhanced Hotel Rate Info](/versions/v5/glossary#enhanced-hotel-rate-info) object with confirmed pricing.

### Possible Errors

| Status Code | Error | Reason |
|---|---|---|
| 404 | [Session Not Found] | The specified session ID does not exist or has expired |
| 404 | [Rate Not Found] | The specified rate ID does not exist or is no longer available |
| 412 | [Search Not Initiated] | No search has been started for this session |
| 410 | [Rate Expired] | The rate has expired and is no longer available for pricing |
| 409 | [Rate No Longer Available] | The rate is no longer available due to inventory changes |
