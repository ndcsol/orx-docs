# Hotel Property Details

This page covers retrieving comprehensive information about a specific hotel property, including detailed descriptions, amenities, policies, and check-in instructions.

## Hotel Property Details

`GET /hotel/[sessionId]/property/[selectedAvailability]`

This endpoint retrieves comprehensive information about a specific hotel property, including detailed descriptions, amenities, policies, check-in instructions, and other property-specific information. The `selectedAvailability` parameter should be the public identifier of a hotel from the availability results.

> ðŸ›Ž Use the `public` ID from a hotel availability result to fetch detailed property information including amenities, policies, and check-in procedures.

### Request

```json filename="GET /hotel/[sessionId]/property/[selectedAvailability]" copy
```

### Possible Errors

| Status Code | Error | Reason |
|---|---|---|
| 404 | [Session Not Found] | The specified session ID does not exist or has expired |
| 404 | [Property Not Found] | The specified property ID does not exist in the current session |
| 412 | [Search Not Initiated] | No search has been started for this session |
