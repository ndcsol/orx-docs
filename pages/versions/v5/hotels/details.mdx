# Hotel Property Details

This page covers retrieving information about a specific hotel property and its rooms.

## Hotel Property Details

`GET /hotel/[sessionId]/property/[propertyId]`

This endpoint retrieves information about a specific hotel property, including detailed descriptions, amenities, policies, check-in instructions, and other property-specific information. The `propertyId` parameter should be the public identifier of a hotel from the availability results.

> ðŸ›Ž Use the `public` ID from a hotel availability result to fetch detailed property information including amenities, policies, and check-in procedures.

### Request

`GET /hotel/[sessionId]/property/[propertyId]`

### Possible Errors

| Status Code | Error | Reason |
|---|---|---|
| 404 | [Session Not Found] | The specified session ID does not exist or has expired |
| 404 | [Property Not Found] | The specified property ID does not exist in the current session |
| 412 | [Search Not Initiated] | No search has been started for this session |
